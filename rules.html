Rules
Squaratar is a site with a 100x100 field. In this field, anyone can occupie a 1x1 square place, the location, color and other properties of which can be determined by him/herself. The occupied place stays that way forever. The occupying must be sent by e-mail.
  • E-mail: squaratar@gmail.com

  • A occupied 1x1 square place contains the following data: coordinates, color, text, link, cursor.

  • E-mail sample:
      Subject: "occupy", [obligatory]
      (in case of modification: "change" [obligatory])

      1. Coordinate: [obligatory]
      2. Color/code: [obligatory]
      3. Text:
       4. Link:
       5. Cursor:

      More maybe later...
    Among the listed points, at least 1. and 2. point must be sent to the email address squaratar@gmail.com.

  • In the case of occupying or changes, we only accept e-mail messages.

  • A user can occupy only one 1x1 place in the 100x100 field. In this place, you can change your data (except the coordinates) at any time with the e-mail address with which you reserved your place.

  • The user can change her pixel at any time (except the coordinates) by e-mail request. In this case, the word "change" must be written in the subject field.

  • We only accept changes to a specific location from the email from which we received the coordinates of the specific location at the time of booking.

  • It is forbidden to use additional e-mail addresses for a user to occupy additional places.

  • We are not accept occupied coordinates.

  • The color can be, for example: red, blue etc. The code for example: HEX, RGB, HSL, CMYK. We not accept transparent color.

  • The text in the text box can be a maximum of 350 characters long, including the spaces. The text can be a maximum of 15 lines (one line ca. 35 chars). The text box must not contain links, images, and etc specials (except emojis and symbol). The text can be: advertisement, about yourself, milestone, memory, request, saying, proverb, demonstration, message to another user using their coordinates or anything...

  • The link is activated by clicking on the pixel. It can be anything, but just one and no virus. We are not responsible for the content of the links.

  • The image of the cursor above the text field can consist of a maximum of 32x32 pieces image points. Less is allowed.

  • The sweepstakes winner's pixel will be automatically adjusted to the amendment specified in the sweepstakes. You can change this at any time.

  • Daily table update.

  • It is not recommended to use this page on Internet Explorer.

  • We undertake not to share a user's private data (except for what the given user asked us to share) with third parties.


If the e-mail does not meet the above, we will ignore it!
