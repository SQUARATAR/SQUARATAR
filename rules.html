Rules
Squaratar is a site with a 100x100 field. In this field, anyone can occupie a 1x1 square place, the location, color and other properties of which can be determined by him/herself. The occupied place stays that way forever. The reservation must be sent by e-mail according to the following rules:
  • A occupied 1x1 square place contains the following data: coordinates, color, text, link, cursor.

  • E-mail sample:
      Subject: "occupie", [obligatory]
      in case of modification: "change" [obligatory]
      I. Coordinate: [obligatory]
      II. Color/code: [obligatory]
      III. Text:
      IV. Link:
      V. Cursor:
      More maybe later...
    Among the listed points, at least I. and II. point must be sent to the email address squaratar@gmail.com.

  • A user can occupy only one place in the 100x100 field. In this place, you can change your data at any time (except the coordinates) with the e-mail you use.

  • The user can change her pixel at any time (except the coordinates) by e-mail request.

  • We only accept changes to a specific location from the email from which we received the coordinates of the specific location at the time of booking.

  • It is forbidden to use additional e-mail addresses for a user to occupy additional places.

  • We are not accept occupied coordinates.

  • Color for example: red, blue. Code for example: HEX, RGB, HSL, CMYK. We not accept transparent color.

  • The text in the text field can be a maximum of 300 characters long, including the spaces and line breaks. The text can be: advertisement, about yourself, milestone, memory, request, saying, proverb, demonstration, message to another user using their coordinates or anything...

  • Link is anything but just one and no virus. We are not responsible for the content of the links.

  • The image of the cursor above the text field can consist of a maximum of 40x40 image points. Less is allowed.

  • The sweepstakes winner's pixel will be automatically adjusted to the amendment specified in the sweepstakes. You can change this at any time.

  • Daily table update.

  • We undertake not to share a user's private data (except for what the given user asked us to share) with third parties.


If the e-mail does not meet the above, we will ignore it!
